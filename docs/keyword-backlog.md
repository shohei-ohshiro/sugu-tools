# ツール候補キーワードバックログ（優先順）

2026-07-04 第1回経営会議の市場調査（SERP実査含む）に基づく。実装前にラッコキーワード等で検索ボリューム実測が望ましい。
原則: 月間検索50〜500の長尾×上位が個人サイト・記事のみの領域を狙う。テーマの塊（建築系・身体系）を優先。

## 作らない/主戦場にしないもの

- 文字数カウント・和暦変換・パーセント計算のビッグキーワードでの上位狙い（老舗・大手が占有。品揃えとして維持のみ）
- 医療・法律・税の正確な助言に踏み込むツール（概算＋免責で書けるものだけ）
