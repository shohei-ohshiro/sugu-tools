# オーナー承認待ちの提案

経営会議が起案し、金銭コストまたはオーナー本人の作業が必要なため承認・実行を待っている案件。
承認されたら該当行を「決定事項」へ移す。

## 提案中

### P-1: 独自ドメイン取得（最優先・年約1,700円）— 2026-07-04 起案

- **なぜ**: vercel.appサブドメインではGoogle AdSenseの審査自体が受けられない（サイト追加がルートドメイン単位のため）。AdSenseに限らずMediavine/Ezoic等の主要広告網もすべて独自ドメイン必須。**これを払わない限り広告収益の天井はほぼゼロ**
- **何を**: `.com` ドメイン1本（例: sugu-tools.com など空いているもの）。Xserverドメイン/お名前.comの1円キャンペーンか、Cloudflare Registrar（約$10/年・原価販売）
- **オーナーの作業**: レジストラでドメイン取得（10分）→ Vercelダッシュボード → sugu-tools → Settings → Domains に追加 → レジストラ側でAレコード 76.76.21.21 を設定
- **その後**: 旧URLから308リダイレクト設定・サイト内URL書き換えは経営会議側で実施

### P-2: Google Search Console 登録（無料・オーナー作業10分）— 2026-07-04 起案

- **なぜ**: 検索流入の計測とsitemap送信。AdSense申請の前提でもある
- **オーナーの作業**: https://search.google.com/search-console にGoogleアカウントでログイン→プロパティ追加（独自ドメイン取得後が効率的）

### P-3: A8.net・もしもアフィリエイト登録（無料・オーナー作業各15分）— 2026-07-04 起案

- **なぜ**: サイト審査ほぼなしで登録でき、ツールに関連する成約型広告の口座を先に開けておける。楽天/Amazon物販はもしも経由が定石
- **急がない**: AdSense承認より優先度低。フェーズ2以降で可
